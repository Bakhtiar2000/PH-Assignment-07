1. Show the differences between map, filter, forEach and find

map filter forEach find
map method is used to iterate through an array and create a new array by performing an operation on each element of the original array filter method is used to iterate through an array and create a new array that contains only the elements that pass a given test forEach method is used to iterate through an array and perform a given operation on each element of the array find method is used to iterate through an array and return the first element that passes a given test

3. What is Event Loop and how does it work?

The event loop is a mechanism in JavaScript that allows for asynchronous operations to be executed in a non-blocking manner. It handles the execution of code in response to events such as user interactions, network requests, and timers.

The event loop works by continuously monitoring the call stack and the message queue.When an event is triggered, such as a user clicking a button or a network request completing, a message is added to the message queue. The event loop continuously checks the call stack and the message queue. If the call stack is empty, it takes the first message from the queue and pushes it onto the call stack. This results in the associated function being executed. If the call stack is not empty, the message remains in the queue until the call stack is cleared.

This process allows JavaScript to handle asynchronous operations without blocking the execution of other code.

This function executes the 60s interval time after we get into the start quiz page. When this function is called without a parameter, it stops the timer by executing clearInterval(). But when called with a parameter, it initializes the timer with setInterval. The time is shown with a combination of minute and second.
